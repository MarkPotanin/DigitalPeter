# DigitalPeter
Data and description for the Digital Peter dataset.

## Data
Data link - https://drive.google.com/file/d/1pl-NVBsjD7lgMYdTx6eFVU1s7rZfeeLY/view?usp=sharing. It consists of images and txt files for cutted text lines.

Full documents - https://drive.google.com/file/d/1CcJpmE0yIJ2IY1PPKyil8ZrOuCfI1Jpv/view?usp=sharing. It consists of images of full documents. You can use it to train models that cut the text lines from initial image.

Annotations - https://drive.google.com/file/d/1XxUMb4_NoNPPKioa4RqDLmSQ0Esu_Fbt/view?usp=sharing. It is annotation for the full documents dataset (previous link). Annotations include segmentation masks for text lines.

## Description

The dataset consists of 9694 images and text files. There are 265788 symbols and approximately 50998 words.
